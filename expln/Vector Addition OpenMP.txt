Important Concepts

1. Vector

A vector is simply an array of elements.

Example:

int A[5] = {1,2,3,4,5};


2. Parallel Programming

Normally CPU executes one task at a time.

In parallel programming, multiple tasks run simultaneously using multiple CPU cores.

Example:

Core 1 adds A[0]+B[0]
Core 2 adds A[1]+B[1]
Core 3 adds A[2]+B[2]

This makes execution faster for large data.


3. OpenMP

OpenMP is a library used for parallel programming in C/C++.

Header file:

#include <omp.h>

OpenMP uses compiler directives like:

#pragma omp parallel for

This tells compiler:

“Run this loop in parallel.”

How Parallelism Works Internally

Without OpenMP:

CPU:
Add 1st element
Add 2nd element
Add 3rd element
...

With OpenMP:

Thread 1 → C[0]
Thread 2 → C[1]
Thread 3 → C[2]
Thread 4 → C[3]

Multiple additions happen together.


Sequential vs Parallel

Sequential

for(int i=0;i<n;i++)
    C[i]=A[i]+B[i];

Single thread.


Parallel

#pragma omp parallel for
for(int i=0;i<n;i++)
    C[i]=A[i]+B[i];

Multiple threads.

Faster for large vectors.
