If:

$$

Formula Used

For every element:

C_{ij}=\sum_{k=0}^{n-1} A_{ik} \times B_{kj}

Meaning:

Take one row from A
Take one column from B
Multiply corresponding elements
Add them

22. Understanding OpenMP

What is OpenMP?

OpenMP = Open Multi-Processing

Used for:

Parallel Programming
Multi-core CPU usage


Sequential vs Parallel

Sequential

One thread does all work.

Thread 1:
Row1 → Row2 → Row3


Parallel

Multiple threads share work.

Thread 1 → Row1
Thread 2 → Row2
Thread 3 → Row3

Faster execution.


23. Why Parallelization Works Here

Each row calculation is independent.

Example:

Row 1 result does not depend on Row 2
So threads can work simultaneously

Perfect for parallel programming.
