What is Bubble Sort?

Bubble Sort is a simple sorting algorithm where:

Adjacent elements are compared
If they are in the wrong order, they are swapped
Largest elements “bubble up” to the end after each pass

Example:

Array:
5 3 1 4

Pass 1:

5 > 3 → swap → 3 5 1 4
5 > 1 → swap → 3 1 5 4
5 > 4 → swap → 3 1 4 5

Largest element 5 reached the end.


What is Sequential Bubble Sort?

Sequential means:

Code runs one step after another
Single thread executes everything

Only one CPU core works.


What is Parallel Bubble Sort?

Parallel means:

Multiple operations happen at the same time
Multiple threads/cores work together

This code uses OpenMP for parallelism.


What is OpenMP?

OpenMP is a library/API used for parallel programming in C/C++.

It allows:

Multi-threading
Faster execution
CPU core utilization

Implements Parallel Bubble Sort using OpenMP.


Odd-Even Transposition Sort

Parallel bubble sort here uses:

Odd-Even Phases

Because normal bubble sort cannot fully parallelize safely.

Two phases:

Even Phase

Compare:

(0,1)
(2,3)
(4,5)

Odd Phase

Compare:

(1,2)
(3,4)
(5,6)

This avoids thread conflicts.


Phase Loop

Even Phase

Checks if phase is even.

% = modulus operator

Returns remainder.


OpenMP Parallel For

Very important line.

This tells OpenMP:

Divide loop iterations among multiple threads.



Multiple comparisons happen simultaneously.


Even Index Comparisons

Odd Phase

(1,2)
