1. What is Merge Sort?

Definition

Merge Sort is a Divide and Conquer sorting algorithm.

It works in 3 steps:

Divide array into 2 halves
Sort both halves recursively
Merge sorted halves


2. What is Parallel Merge Sort?

Definition

Parallel Merge Sort performs sorting of different parts of the array simultaneously using multiple threads.

Your code uses OpenMP sections for parallelism.
