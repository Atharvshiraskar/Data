This project is a Deep Learning based Sentiment Analysis System using the TensorFlow and IMDB movie reviews dataset.

1. Basic Definitions (Very Important for Viva)

What is Sentiment Analysis?

Sentiment Analysis means identifying emotions/opinions from text.

Example:

“Movie was amazing” → Positive
“Worst movie ever” → Negative


2. Important Concepts & Definitions

Deep Learning

Deep Learning is a subset of Machine Learning that uses Neural Networks with multiple layers to learn patterns automatically.


Neural Network

A Neural Network is a system inspired by the human brain that learns from data using interconnected neurons.


CNN (Convolutional Neural Network)

CNN is mainly used for feature extraction.

In this project:

CNN finds important words/patterns in reviews.

Example:

“excellent acting”
“waste of time”

These patterns help classify sentiment.


LSTM (Long Short-Term Memory)

LSTM is a type of Recurrent Neural Network (RNN).

It remembers previous words and understands sentence sequence.

Example:

“The movie was not good”



LSTM understands that “not” changes the meaning.


Embedding Layer

Embedding converts words into numerical vectors.

Computer cannot understand text directly, so words are converted into numbers.

Example:

good -> [0.12, 0.45, ...]
bad  -> [-0.22, 0.91, ...]


Activation Function

Activation function decides whether a neuron should activate or not.

ReLU

activation='relu'

Formula:

f(x)=\max(0,x)

Negative values → 0
Positive values → unchanged

Used because:

Fast
Solves vanishing gradient problem


Sigmoid Function

Used in output layer for binary classification.

Formula:

\sigma(x)=\frac{1}{1+e^{-x}}

Output range:

0 to 1

Example:

0.9 → Positive
0.1 → Negative


Loss Function

Loss function measures model error.

This project uses:

binary_crossentropy

Because:

Only 2 classes (positive/negative)

Lower loss = better model.


Optimizer

Optimizer updates weights to reduce loss.

This project uses:

adam

Adam optimizer is:

Fast
Efficient
Most commonly used


Epoch

1 Epoch = One complete pass of training data through model.

Example:

EPOCHS = 10

Means model learns from dataset 10 times.


Batch Size

Batch size = Number of samples processed at once.

BATCH_SIZE = 128

Means:

128 reviews processed together.

Viva Questions & Answers

Why CNN is used?

CNN extracts important features and patterns from text.


Why LSTM is used?

LSTM understands sequence and context of words.


Why sigmoid activation?

Because output is binary classification.


Why binary crossentropy?

Because only two classes exist.


Why dropout layer?

To prevent overfitting.


Difference between training and testing data?

Training data → used for learning
Testing data → used for final evaluation


What is overfitting?

When model memorizes training data but performs poorly on new data.


What is padding?

Padding makes all sequences equal length.


Why Adam optimizer?

Adam is fast and efficient for deep learning.
